# NestJS Pokemon API

* Docker
* MongoDB
* NestJS v10


### Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar 

``` 
npm install 
```

3. Instalar NestJS CLI

``` 
npm install -g @nest/cli 
```

4. Levantar contenedor de MongoDB

``` 
docker compose up -d 
```
